# Student Study Group Finder - Milestone 5

A full-stack web application built for UCU students to find, join, and create study groups.

## 🚀 Features
- **User Profiles:** Personalized student dashboards with Program and Year of Study.
- **Group Discovery:** Search for groups by Course Code or Faculty.
- **Admin Dashboard:** System stats (Total Users/Groups) for administrators.
- **Secure Auth:** JWT-based authentication with bcrypt password hashing.

## 🛠️ Tech Stack
- **Frontend:** React.js, Axios, React Router
- **Backend:** Node.js, Express.js
- **Database:** MySQL

## ⚙️ How to Run Locally
1. **Clone the repo:** `git clone https://github.com/wandwasibashir-arch/student-study-group-finder.git`
2. **Setup Database:** Import the provided SQL schema into MySQL.
3. **Backend:** 
   - `cd backend`
   - `npm install`
   - `node app.js`
4. **Frontend:**
   - `cd frontend`
   - `npm install`
   - `npm start`
